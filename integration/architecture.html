⏱️ Trade Lifecycle Timeline

⏰ Every 30 minutes
🧠 Learning Server scans for opportunities
Fetches 15m / 1h / 4h candles for all active pairs. Computes RSI, Bollinger Bands, EMA, ADX. Looks for confluence: BB touch + RSI extreme + trend alignment across timeframes.
🎯 Signal ≥ 70% confidence
🤖 Signal Executor checks risk gates
Circuit breaker active? Balance > $5? Max positions reached? Already in this pair? If all clear, calculates position size (2% of $balance × 5x leverage) and sends to trading server.
⚡ POST /api/execute
💹 Trading Server places market order
Sends MARKET order to Binance Futures. Records position with entry price, quantity, confidence, and reason. Increments daily trade counter.
📡 While position is open
🛡️ Risk Manager monitors position
WebSocket streams real-time price updates. ATR-based stop loss, break-even after 1:1 reward, trailing stop on winners. Circuit breaker tracks daily PnL.
🔚 Position closes
📊 Learning Server analyzes result
Trade result written to trade_results table. Shadow Analyzer profiles holding time, entry/exit timing, detects behavioral biases (disposition effect, overtrading, chase entries, anchoring).
🔄 Weekly (Monday 12:00)
🔬 Pair Universe rotated
Suitability Scorer re-evaluates all 29 pairs on movement quality. Bottom performers replaced. Config auto-updated and trading server hot-reloaded.
